# Reddit Video Downloader

A fast, clean, and free web application to easily download any Reddit video. Built with React, Vite, TypeScript, and styled with Tailwind CSS + shadcn/ui.

## ✨ Features
- **Lightning Fast**: Optimized video fetching and parsing.
- **No Login Required**: Instantly download videos without creating an account.
- **Works Everywhere**: Fully responsive design for mobile, tablet, and desktop.
- **High Quality MP4s**: Extracts the highest available quality, often with audio merged.

## 🚀 Getting Started

### Prerequisites
Make sure you have Node.js and npm installed.

### Installation & Running Locally

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

## 🧪 Testing with TestSprite

This project includes AI-generated tests via [TestSprite](https://www.testsprite.com/). Tests are located in the `testsprite_tests` directory.

To test the application:
1. Ensure the TestSprite MCP server is configured in your IDE.
2. The `testsprite_tests` directory contains comprehensive unit, component, and integration tests for the core logic (`redditApi.ts`) and React components (`UrlInput`, `VideoPreview`, `ErrorMessage`, `Features`).
